# PHP Crash Course - Learning Journey

Working through **PHP Crash Course: The Complete, Modern, Hands-On Guide** by Matt Smith (No Starch Press, 2024).

## 📚 About This Repository

This repository tracks my progress learning modern PHP web development through a comprehensive, hands-on approach. Each section builds practical skills for creating professional, secure web applications.

## 🎯 Learning Objectives

- Master modern PHP fundamentals and best practices
- Build secure, database-driven web applications
- Implement professional features (authentication, shopping carts, sessions)
- Apply object-oriented programming and design patterns
- Use industry-standard tools (Composer, Twig, PDO)
- Structure applications using MVC architecture
- Write clean, maintainable, production-ready code

## 🛠️ Development Environment

### Primary Environment
- **Replit** (https://replit.com) - Cloud-based IDE with PHP support
- Configured per Appendix C of the book
- No local installation required

### Alternative: Local Setup
- **PHP Version:** 8.x (latest stable)
- **Database:** MySQL/MariaDB or SQLite
- **Editor:** VS Code with PHP extensions
- **Tools:** Composer (dependency management)

### Online Alternatives
- **PHPSandbox.io** - Full PHP IDE with project support
- **OnlinePHP.io** - Quick testing and prototyping
